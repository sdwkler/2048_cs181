Pattern (absolute) is precise for seen states; Diff (relative) encodes shape-similar boards identically → generalizes to unseen positions. Complementary.

5. Bitboard Engine: 64-bit Integer + 65536-entry LUT = O(1) Slide

Board Encoding: 16 cells → one 64-bit integer

graph TD ROW0["Row 0: 4 cells x 4bit = 16bit"] ROW1["Row 1: 4 cells x 4bit = 16bit"] ROW2["Row 2: 4 cells x 4bit = 16bit"] ROW3["Row 3: 4 cells x 4bit = 16bit"] ENCODED["64-bit integer
board = row0 lshift 48 | row1 lshift 32 | row2 lshift 16 | row3
Each cell stores exponent: 0=empty, 1=2, 2=4, ... 15=32768"] ROW0 --> ENCODED ROW1 --> ENCODED ROW2 --> ENCODED ROW3 --> ENCODED style ROW0 fill:#e0e7ef,stroke:#7a8faa style ROW1 fill:#e0e7ef,stroke:#7a8faa style ROW2 fill:#e0e7ef,stroke:#7a8faa style ROW3 fill:#e0e7ef,stroke:#7a8faa style ENCODED fill:#e3efe0,stroke:#4a8a4a,stroke-width:2px

4 rows × 4 cols × 4bit = 64bit. One integer = full board state. Shift + mask extracts any row/cell.

Slide Pipeline: Pure LUT O(1)

graph LR DIR["Direction unify
UP/DOWN/RIGHT
→ rotate to LEFT"] EXTRACT["Extract row_i
shift + mask"] LUT["row_lut 65536
single array access"] MERGE["OR into output
accumulate score"] DONE["Slide complete
4 lookups total"] DIR --> EXTRACT --> LUT --> MERGE --> DONE style DIR fill:#f5f1ec,stroke:#d5cec4 style LUT fill:#e3efe0,stroke:#4a8a4a,stroke-width:2px style MERGE fill:#e0e7ef,stroke:#7a8faa style DONE fill:#e3efe0,stroke:#4a8a4a

No per-cell loops. No conditional branches. All merge decisions precomputed at startup — runtime is pure integer ops + memory reads. Microsecond-level per move.
